Что такое Windows Defender?

Windows Defender (ныне Microsoft Defender Antivirus) — это встроенное средство защиты от вирусов и вредоносного ПО, доступное на системах Windows начиная с Windows 8.

Основная функция — обеспечение **базовой защиты в реальном времени** без необходимости установки стороннего антивируса.

История создания

Windows Defender был создан корпорацией **Microsoft** в 2006 году, как продолжение приобретённой программы **GIANT AntiSpyware**. Впервые появился в Windows XP.

Начиная с **Windows 8**, Defender стал полноценным антивирусом и предустановлен в систему.

Как включить или выключить Windows Defender?

**Чтобы включить Windows Defender:**

  1. Откройте **"Параметры"** → **"Обновление и безопасность"**.
  2. Перейдите в раздел **"Безопасность Windows"**.
  3. Включите параметры защиты в реальном времени.

**Чтобы временно выключить:**

  • Перейдите в те же настройки.
  • Отключите "Защиту в реальном времени".
  • Будьте осторожны: это снижает защиту вашего устройства.

Основные функции Windows Defender

  • Защита от вирусов и вредоносного ПО в реальном времени.
  • Проверка скачиваемых файлов и приложений.
  • Родительский контроль и защита сети.
  • Мониторинг производительности устройства.
